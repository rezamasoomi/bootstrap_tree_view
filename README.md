# optional_tree_view
## this is an tree view with this fuature: 
    - checkable all items 
    - checkable just last chiled level items 
    - disabled default items
    - checked default items
    - multi level
    - multi choice or single choice 
    - rtl and ltr
    - right or left position of checkbox
